# Benqi-subgraph

### Commands:

generate types:
 - `npx graph codegen --output-dir src/types/`
build code:
 - `npx graph build`
deploy:
 - `npx graph deploy --debug --access-token <TOKEN> yhayun/Experiment --node https://api.thegraph.com/deploy/ --ipfs https://api.thegraph.com/ipfs/`
